# Analysis pipeline
 
This repository contains the analyses scripts for the AdV infection paper. The scripts for the analysis of ONT full-length reads can be found [here](https://github.com/nanoporetech/pipeline-transcriptome-de). The repository for ONT basecalling using using enroot containers on the [LRZ AI systems](https://www.lrz.de/) cloud can be found [here](https://github.com/hcstubbe/docker_seq).
